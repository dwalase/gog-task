# Gog Frontend Recruitment Task - Daniel Walasek

## Installation

To install dependencies, use:

```sh
npx npm install
```

## Run tasks

To run the dev server use:

```sh
nx run gog-task:serve
```

To start Storybook with UI elements:

```sh
nx run ui:storybook
```

To run e2e tests use:

```sh
nx run gog-task-e2e:e2e
```
